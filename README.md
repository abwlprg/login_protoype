# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). The front end was created with the use of Create-React-App.

## Available Scripts

In the project directory, you can run:

### SERVER SIDE `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

You have to make a [config.env] file for in the root directory. This will start the server and has all functionalities that were requested.

The values that need to be made are :
PORT = 
MONGO_URI = ""

JWT Stuff
JWT_SECRET=""
JWT_EXPIRE=1""

SENDGRID STUFF
EMAIL_SERVICE=SendGrid
EMAIL_USERNAME=''
EMAIL_PASSWORD=	''
EMAIL_FROM=''


### CLIENT SIDE `npm start`

A simple 'npm start' will start up the [http://localhost:3000] and showcase a framework for a login that was made using React.
